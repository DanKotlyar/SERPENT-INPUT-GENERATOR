User's Manual
=============

The User\'s Manual for the ``SIMBA`` package describes how 
to use the different containers and function to easily add, edit, and retrieve data. 

Containers
-------------

Multiple containers exist to enable easy addition and edit of data

	1. :ref:`Material Container <material_cont>`: define a single or multiple ``TransmutationData`` containers.
	2. :ref:`Material Container New <materialNew_cont>`: define a single or multiple ``TransmutationData`` containers.
	2. :ref:`Material Container New New <modules>`: define a single or multiple ``TransmutationData`` containers.
